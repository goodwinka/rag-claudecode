# Инструкции для работы с базой знаний

У тебя есть доступ к локальной базе знаний по программированию.
Содержимое: C, C++, Python, Qt, алгоритмы, структуры данных,
форматы файлов, системы сборки, сетевые протоколы.

## Инструменты

- `search_docs` — общий поиск по документации, API, паттернам
- `search_api` — поиск по сигнатурам функций и классов
- `search_algorithm` — поиск алгоритмов и структур данных
- `search_qt` — поиск по Qt
- `search_technical` — форматы файлов, системы сборки, протоколы

## Когда ОБЯЗАТЕЛЬНО вызывать поиск

1. Пишешь код с библиотекой/фреймворком — `search_api`
2. Используешь Qt классы — `search_qt`
3. Реализуешь алгоритм — `search_algorithm`
4. Работаешь с форматом файлов или сборкой — `search_technical`
5. Не уверен в сигнатуре или паттерне — `search_docs`
6. Работаешь с памятью C/C++ — `search_docs("memory management")`
7. Обрабатываешь ошибки — `search_docs("error handling", language="...")`

## Правила

- Лучше лишний поиск, чем неправильный код
- Если база знаний и твои знания конфликтуют — доверяй базе
- При работе с C/C++ ВСЕГДА проверяй правильность управления памятью
- При работе с Qt ВСЕГДА проверяй жизненный цикл объектов
